# VRHost Lab

> Lightweight network lab platform for building and managing virtual router topologies

Built by a NOC engineer for NOC engineers. Focus on simplicity, speed, and CLI integration.

## Features (Planned)
- [ ] Web UI for router management
- [ ] Integration with Juniper vSRX
- [ ] Visual topology builder
- [ ] Web-based console access
- [ ] Lab templates and saving
- [ ] API-first design

## Tech Stack
- Backend: Python FastAPI + libvirt
- Frontend: React + Cytoscape.js
- Platform: Ubuntu/KVM
